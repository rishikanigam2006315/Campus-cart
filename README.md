# 🛒 Campus Cart Backend

Backend service for the **Campus Cart Android App**.
Built using Spring Boot to handle authentication, product listings, image uploads, user profiles, and notifications.

## 🚀 Features

* 🔐 JWT-based Authentication (Login / Signup)
* 🛍️ Product CRUD APIs (Buy & Sell system)
* 📸 Image Upload using **Cloudinary**
* 👤 User Profile with profile image support
* ❤️ Favorites / Wishlist system
* ⭐ Seller Rating system
* 🔔 Push Notifications support (FCM token storage)
* 🔎 Search, filter, and pagination APIs

## 🧱 Tech Stack

* Java (Spring Boot)
* Spring Security (JWT)
* Spring Data JPA
* PostgreSQL (Supabase)
* Cloudinary (Image storage)
* Firebase Cloud Messaging (FCM)
* Render (Deployment)
